# locksmith_app
### Introduction
This is a mobile application project using react native for Fontend and Nodejs for the Backend. The project consists of two applications, one for the customer and one for the locksmith. Support customers to order online lock repair services.
### Server
[backend locksmith](https://github.com/hocnv21/BackEndLocksmith.git) is used as the backend for this app.
### Features
This application provides users with the following features
* Authentication using Firebase
* Order online lock repair services(for user app)
* View order history(for use app)
* Income statistics(for worker app)
* Map directions(for worker app)
